# PhotoVibe Android App

사진을 선택하고, 프리셋 느낌을 적용하고, 인스타/릴스/틱톡/웹에 맞는 크기로 저장·공유하는 안드로이드 앱 초안입니다.

## 포함 기능
- 갤러리에서 사진 선택
- 프리셋: 원본, 아이폰 자연톤, 갤럭시 선명톤, 인스타 쨍한톤, 시네마틱, 따뜻한 필름, 빈티지, 흑백
- 사이즈: 원본비율, 인스타 정사각 1080, 인스타 세로 1080x1350, 릴스/틱톡 1080x1920, 웹/썸네일 1280x720
- 갤러리 저장
- 카카오톡 공유
- 인스타그램 공유

## APK 만드는 법
1. Android Studio 설치
2. 이 폴더를 Open
3. Gradle 동기화 완료 후 Build > Generate App Bundles or APKs > Generate APKs
4. 생성된 APK를 안드로이드 폰으로 옮겨 설치
